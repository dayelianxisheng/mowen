# Mowen 机器人仿真启动命令参考

## 环境准备

```bash
# 进入容器
docker exec -it mowen_sim bash

# 每次新终端都要 source
source /opt/ros/humble/setup.bash
source /root/mowen_ws/install/setup.bash
```

## 1. 启动 Gazebo 仿真世界（终端1）

```bash
ros2 launch mowen_gazebo mowen_world.launch.py
```

## 2. 启动 Cartographer SLAM（终端2）

```bash
ros2 launch mowen_cartographer cartographer.launch.py use_sim_time:=true
```

启动后会同时打开 RViz2。

## 3. 遥控小车（终端3）

```bash
ros2 run teleop_twist_keyboard teleop_twist_keyboard --ros-args -r cmd_vel:=/cmd_vel
```

控制键：`i`前进 `,`后退 `j`左转 `l`右转 `u`左前 `o`右前 `k`停止

## 4. 保存地图

```bash
# 先结束轨迹（必须）
ros2 service call /finish_trajectory cartographer_ros_msgs/srv/FinishTrajectory '{trajectory_id: 0}'

# 保存地图
ros2 run nav2_map_server map_saver_cli -f /root/mowen_ws/map
```

## 5. 常用调试命令

```bash
# 查看话题列表
ros2 topic list

# 查看 scan 数据
ros2 topic echo --once /scan

# 查看地图发布频率
ros2 topic hz /map

# 查看 TF 树
ros2 run tf2_ros tf2_echo odom base_link

# 查看子图列表
ros2 topic echo --once /submap_list
```

## 6. 从宿主机拷贝地图

```bash
docker cp mowen_sim:/root/mowen_ws/map.pgm ./
docker cp mowen_sim:/root/mowen_ws/map.yaml ./
```

## 7. 生成场景

```bash
python3 /root/mowen_ws/src/mowen/scripts/gen_mapping_world.py
```

## 8. 修改模型后重建

```bash
cd /root/mowen_ws
colcon build --symlink-install
```
